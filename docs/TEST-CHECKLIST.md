## 附录: 快速回归脚本

```bash
# 一站式门禁(15-20 分钟)
npm run validate

# Web E2E(10-15 分钟)
npm run test:e2e

# Electron E2E(15-20 分钟)
npm run test:e2e:electron-all

# 覆盖率
npm run test:coverage

# 变异测试(30 分钟,可选)
npm run test:mutation

# 性能回归
npm run perf

# Smoke 测试
npm run smoke
```
