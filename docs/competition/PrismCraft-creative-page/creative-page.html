/ 创作流水线

端到端创作流

从故事灵感到视频成片，四步走完整创作链路，每一步都有 AI 辅助。

01
故事规划 LLM
输入故事标题、类型、基调，AI 自动拆解为结构化分镜（StoryBeat），每个镜头绑定角色、场景、道具。
02
关键帧生成 Image Model
为每个镜头生成关键帧画面，支持首尾帧对（Frame Pair）生成，自动注入角色/场景元素绑定信息保证一致性。
03
视频生成 Video Model
基于关键帧 + Prompt 生成视频片段，支持首尾帧驱动、参考图加权、批量生成，任务队列后台轮询。
04
一致性校验 Reference Engine
跨镜头角色/场景一致性检查，引用引擎自动比对，避免"换脸"和"场景漂移"。
/ 核心能力

四大核心设计

🔌
多模型统一接入
13+ AI 提供商通过统一接口接入。声明式 JSON 插件实现零代码新增模型，4 层能力解析（插件配置 → 精确匹配 → 前缀模糊 → 保守默认）自动适配模型差异。
🎬
专业分镜系统
三级生成流水线：故事规划 → 镜头生成 → 引用检查。每个 Beat 绑定角色/场景/道具，Prompt 引擎自动注入元素信息，LLM 规划结果经 Zod Schema 校验。
🏗️
AI 可维护性工程
零依赖 shared-logic 层双向复用、DI 容器管理有状态服务、contract.json 约束模块边界。让 AI 编程工具用最少上下文做最正确决策。
🔒
本地优先 & 数据隐私
Electron 桌面应用，数据存储在本地 SQLite。API 密钥使用系统级加密（Windows Credential Manager / macOS Keychain），HTTP API + IPC 双通道文件操作。
